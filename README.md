#klvim
=====

My vim settings/plugins

##How to use


$ git clone git://github.com/vklepikovskiy/klvim.git ~/klvim

$ ln -s ~/klvim/vimrc ~/.vimrc

$ ln -s ~/klvim/vim ~/.vim
