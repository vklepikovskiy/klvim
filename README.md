#klvim
=====

My vim settings/plugins

##How to use

Run in shell:

$ git clone git://github.com/vklepikovskiy/klvim.git ~/klvim

$ ln -s ~/klvim/vimrc ~/.vimrc

$ ln -s ~/klvim/vim ~/.vim

$ git submodule init

$ git submodule update

Run in VIM:

:Helptags
